# Web Search Skill

Use this skill when the user asks for current information, recent developments, or facts that are likely to change over time.

## Instructions

1. Use the `web_search` tool to retrieve current information.
2. Base the answer on the search result payload.
3. Summarize the result concisely.
4. Include citations or source URLs when they are present in the tool output.
